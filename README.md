# koydo-comptia-security

CompTIA Security+ exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM154 |
| Name | CompTIA Security+ |
| Country | Global |
| Category | Professional certification |
| Tier | Tier 5 |
| Wave | Wave 7 |
